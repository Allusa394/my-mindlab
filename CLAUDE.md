# MindLab — CLAUDE.md

## Что это за проект
AI-ассистент интеграции знаний из тренингов. Пользователь говорит или вводит инсайт → получает 6-блочный анализ (Смысл, Связь с собой, Осознание, Якорь, Интеграция, Нейроконтур).

Основной сценарий: "слушаю тренинг Ханта → появилась мысль → нажал 'Говорить' → сказал мысль → получил анализ".

## API ключи
- **Groq** — основной AI (работает, бесплатно, llama-3.3-70b-versatile)
- **YandexGPT** — заблокирован (аккаунт заблокирован, нужна привязка карты Россельхозбанка)
- **OpenRouter** — резерв (не работает без VPN)
- Fallback на bank{} если Groq недоступен

## Архитектура AI
1. При загрузке страницы: `fetch('knowledge_base.md')` → загружает 28KB базы знаний
2. При анализе: SYSTEM_PROMPT (инструкции) + knowledge_base.md (контент ТП) + инсайт пользователя
3. Groq отвечает JSON с 6 блоками
4. Если Groq недоступен → fallback на bank{} (локальные ответы)

## Методология — ключевые концепции ТП
- **Воззрение** — способ видения реальности без разделения я/мир
- **Контент** — всё что заполняет поле внимания, формирует программы
- **Мысль не моя** — мысль возникает в пространстве, твоей становится при отождествлении
- **Медитация случается** — нельзя делать из усилия эго (духовный конслагер)
- **Эксперимент Либета** — мозг решает за 0.35 сек до осознания
- **Не забываться в персонаже** — не отождествляться с автоматическими реакциями
- **Осознаёшься пространством** — переживание себя как пространства

## Важные решения
- Всё в одном HTML файле — нет сборки, нет зависимостей
- localStorage для хранения заметок
- knowledge_base.md загружается отдельно через fetch (легко обновлять без изменения index.html)
- Ключи хранятся прямо в index.html (браузерное приложение без сервера)
