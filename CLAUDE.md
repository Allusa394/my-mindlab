# MindLab — CLAUDE.md

## Что это за проект
AI-ассистент интеграции знаний из тренингов. Пользователь говорит или вводит инсайт → получает 6-блочный анализ (Смысл, Связь с собой, Осознание, Якорь, Интеграция, Нейроконтур).

Основной сценарий: "слушаю тренинг Ханта → появилась мысль → нажал 'Говорить' → сказал мысль → получил анализ".

## Архитектура AI

### Источник знаний
- **Встроенная переменная** `knowledgeBase` в index.html (1.6MB)
- Содержит: 14 разделов дистилляции + 4 полных видео Потоков 3.1-3.3 + Структура Сознания
- Загружается при открытии приложения (нет fetch запроса)

### Процесс анализа
1. Пользователь вводит инсайт (текст или голос)
2. SYSTEM_PROMPT (инструкции) + knowledgeBase (полная методология) + инсайт
3. **Groq API** (llama-3.3-70b-versatile) анализирует и возвращает JSON с 6 блоками:
   - СМЫСЛ, СВЯЗЬ С СОБОЙ, ОСОЗНАНИЕ, ЯКОРЬ, ИНТЕГРАЦИЯ, НЕЙРОКОНТУР
4. Если Groq недоступен → fallback на bank{} (локальные шаблонные ответы)

### Обновление knowledge_base
- Редактировать: `knowledge_base.md`
- Синхронизировать с index.html: `node update-kb.js`
- Затем: `git add`, `git commit`, `git push`, `vercel --prod`

## Методология — ключевые концепции ТП
- **Воззрение** — способ видения реальности без разделения я/мир
- **Контент** — всё что заполняет поле внимания, формирует программы
- **Мысль не моя** — мысль возникает в пространстве, твоей становится при отождествлении
- **Медитация случается** — нельзя делать из усилия эго (духовный конслагер)
- **Эксперимент Либета** — мозг решает за 0.35 сек до осознания
- **Не забываться в персонаже** — не отождествляться с автоматическими реакциями
- **Осознаёшься пространством** — переживание себя как пространства
